## Overview

Options Calculator Pro is a comprehensive tool for analyzing calendar spreads and other options strategies. It features:

- Advanced Monte Carlo simulations using Heston stochastic volatility
- Machine learning predictions for trade outcomes
- Professional risk management with Kelly Criterion position sizing
- Real-time market data integration
- Comprehensive backtesting capabilities
